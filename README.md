chrisbot
========

####acts_as_chris
A skype bot to automatically reply to automatically let everyone know I already ate lunch.

####Installation
Download

    git clone https://github.com/jlorich/chrisbot
    bundle install
    
Make chrisbot executable 

    chmod +x ./chrisbot/chrisbot.rb

####Usage

    ./chrisbot.rb
